# MonitorWydatkow
1.	Opis funkcjonalności aplikacji:
- Podział wydatków na kategorie 
  Użytkownik może przypisywać wydatki do kategorii (np. jedzenie, transport, rozrywka, rachunki). Każda kategoria może mieć własną ikonę i kolor. 
- Wyświetlanie statystyk (miesięcznych, tygodniowych, dziennych)
   Aplikacja automatycznie analizuje wydatki i prezentuje dane w formie wykresów kołowych lub słupkowych. Użytkownik może przeglądać historię i trendy. 
- Generowanie raportów (PDF, DOCX) 
  Użytkownik może wygenerować raport wydatków za wybrany okres i pobrać go w formacie PDF lub DOCX. Raport zawiera sumy, kategorie i wykresy. 
- Notatki dla wydatków 
  Możliwość dodania krótkiej notatki do konkretnego wydatku (np. „obiad z przyjaciółmi” lub „prezent urodzinowy”).
- Wiele walut z aktualnymi kursami 
  Aplikacja pobiera aktualne kursy walut z API (np. NBP lub ExchangeRatesAPI) i automatycznie przelicza wartości między walutami. 
- Przypomnienia o płatnościach 
  Użytkownik może ustawić przypomnienia o stałych płatnościach (np. abonament, czynsz, rata kredytu). System powiadomień Androida poinformuje go o terminie.
